Ontology Specification Draft

Abstract

The Building Material Performance ontology is created to enrich building information models with highly aggregated information with respect to material information modelling and material simulation. BMP classifies construction materials based on categories, types and property groups. Each property group describes parameters to be used as input values for simulation works.
